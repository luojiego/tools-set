# 加密工具箱

一个基于 React 的现代化加密工具集合网站，提供密码生成、文本加密解密、身份证校验等实用功能。

## 功能特性

### 🔐 密码生成器
- 支持自定义密码长度（4-50位）
- 多种字符组合选项：
  - 小写字母 (a-z)
  - 大写字母 (A-Z)
  - 数字 (0-9)
  - 特殊字符 (!@#$...)
- 隐藏选项：
  - 纯数字密码
  - 纯字母密码
- 实时密码强度评估
- 一键复制功能
- 密码显示/隐藏切换

### 🛡️ 加密解密工具
- 支持多种加密算法：
  - Base64 编码/解码
  - URL 编码/解码
  - HTML 编码/解码
  - MD5 哈希
  - SHA256 哈希
  - 凯撒密码
  - 替换密码
- 实时算法说明
- 错误处理和提示
- 结果一键复制

### 🆔 身份证校验
- 18位身份证号码格式验证
- 校验位计算和验证
- 详细信息解析：
  - 归属地区
  - 出生日期和年龄
  - 性别判断
  - 校验位验证
- 实时验证反馈
- 详细错误提示和规则说明

## 技术栈

- **前端框架**: React 18
- **构建工具**: Vite
- **UI 组件**: shadcn/ui
- **样式框架**: Tailwind CSS
- **图标库**: Lucide React
- **动画库**: Framer Motion

## 本地开发

### 环境要求
- Node.js 18+
- pnpm (推荐) 或 npm

### 安装依赖
```bash
cd crypto-tools
pnpm install
```

### 启动开发服务器
```bash
pnpm run dev
```

访问 http://localhost:5173 查看应用

### 构建生产版本
```bash
pnpm run build
```
